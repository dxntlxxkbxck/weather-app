# Приложение Погода (Weather App)

## Описание проекта

Курсовая работа - веб-приложение для отображения информации о погоде. Приложение позволяет просматривать текущую погоду, температуру, влажность, скорость ветра и прогноз на несколько дней вперёд для любого города мира

## Технологии

- **HTML5** - структура и разметка приложения
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (ES6+)** - функциональность и работа с API
- **OpenWeatherMap API** - источник данных о погоде (работает только с VPN)
- **Boxicons** - набор иконок для визуализации погодных условий

## Функциональность

**Текущая погода** - отображение информации о погоде на текущий день:
  - Температура
  - Описание погодных условий
  - Иконка, соответствующая погоде
  - Дата и день недели
  - Местоположение (город, страна)

**Дополнительная информация**:
  - Процент осадков
  - Влажность воздуха
  - Скорость ветра

**Прогноз на несколько дней** - список прогноза на следующие дни с температурой и типом погоды

**Поиск по локации** - возможность вводить название города и получать прогноз для этого места

**Определение местоположения** - кнопка для автоматического определения текущего местоположения

## Как использовать

1. Откройте файл `index.html` в любом веб-браузере
2. Приложение загрузит текущую погоду для Санкт-Петербурга
3. Нажмите на кнопку поиск локации, чтобы узнать погоду в других городах

## API и ключи

Приложение использует бесплатный API от **OpenWeatherMap**. Для работы необходимо:
- Действительный API ключ от OpenWeatherMap
- Подключение к интернету

## Особенности реализации

- Использование `fetch API` для получения данных о погоде
- Динамическое обновление иконок погоды в зависимости от условий
- Карта соответствия кодов погодных условий иконкам Boxicons
- Форматирование даты с учётом локализации (русский язык)
- Преобразование температуры в целые числа для удобства чтения
- Адаптивный дизайн для различных размеров экранов
