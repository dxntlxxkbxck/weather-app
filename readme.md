# Приложение Погода (Weather App)

![Превью](src/preview.gif)

## Описание проекта

**Курсовая работа** — веб-приложение для отображения погоды любого города мира. Показывает текущую погоду, детали, прогноз на 4 дня и динамически меняет фон по сезону+погоде.

## Технологии

- **HTML5** — структура разметки
- **CSS3** — адаптивный дизайн + анимации фона  
- **JavaScript ES6+** — вся логика + DOM
- **localStorage** — история поиска (10 городов)
- **OpenWeatherMap API** — данные погоды
- **Boxicons** — иконки погоды

### Архитектура
1. fetchWeatherData() — главная функция
2. changeSeasonalBackground() — анимация фона (4 таймаута)
3. localStorage — история поиска (CRUD)
4. DOM обновления — селекторы + innerHTML
5. Валидация — regex + обработка ошибок API

### Анимация фона:
0.3s → blur(3px) + scale(1.02)
0.6s → blur(0px) + brightness(1.05)  
1.3s → нормальное состояние
